# Simulador de Algoritmos de Memoria

Un simulador interactivo de gestión de memoria que implementa el algoritmo de **Peor Ajuste (Worst Fit)**.

## ✨ Características

- **Algoritmo implementado**: Peor Ajuste (Worst Fit)
- **Visualización interactiva** de la memoria
- **Demos predefinidas** para testing
- **Estadísticas en tiempo real** (uso, fragmentación, bloque mayor libre)
- **Historial de operaciones**
- **Interfaz responsive** que funciona en desktop y móviles

## 🎯 Cómo usar

1. **Agregar procesos**: Especifica el tamaño y haz clic en "Agregar"
2. **Liberar procesos**: Usa "Liberar Aleatorio" para liberar memoria
3. **Ejecutar demos**: Prueba las secuencias predefinidas
4. **Cambiar tamaño**: Modifica el tamaño total de memoria

## 🛠️ Tecnologías

- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript (ES6+)
- GitHub Pages para hosting

## 📊 Algoritmo

El **Peor Ajuste** busca el bloque libre más grande disponible para asignar un proceso, con el objetivo de dejar bloques grandes utilizable para procesos futuros.
